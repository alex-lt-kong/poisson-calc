# Poisson Calculator

A web-based tool that converts an observed probability of an event occurring within a time window into an annualized frequency, using the Poisson distribution.

**How it works:** You provide a time range, a window duration (days/hours), and a probability (e.g., 30%). The calculator tells you how many times per year that event is expected to happen.

**Stack:** FastAPI backend with REST API, plain HTML/CSS/JS frontend. Token-based auth.
